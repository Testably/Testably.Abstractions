# Drive management

> Source: [Examples/DriveManagement](https://github.com/Testably/Testably.Abstractions/tree/main/Examples/DriveManagement)

This example illustrates how to define multiple drives and use space management on individual drives.

The test project shows several common scenarios:

- Add and use an additional drive in the `MockFileSystem`
- Change the default drive (or directory) used for relative paths
- Define and limit the maximum total size of a drive

When the drive's total size limit is exhausted, an `IOException` is thrown for all file operations that write file content.

See [Testing › Drive management](../testing/drive-management) for the conceptual overview.
