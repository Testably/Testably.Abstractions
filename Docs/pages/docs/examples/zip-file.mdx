# Zip-File example

> Source: [Examples/ZipFile](https://github.com/Testably/Testably.Abstractions/tree/main/Examples/ZipFile)

This example shows how to use `IFileSystem` to compress and uncompress directories using `System.IO.Compression`.

The `ZipFileHelper` class provides two methods:

- `Stream CreateZipFromDirectory(string directory)` — creates a zip from all files and sub-directories in the given directory and returns a stream that can be stored in the file system:

  ```csharp
  using (Stream zipStream = ZipFileHelper.CreateZipFromDirectory(directory))
  {
      using FileSystemStream fileStream = FileSystem.File.Create("test.zip");
      zipStream.CopyTo(fileStream);
  }
  ```

- `void ExtractZipToDirectory(Stream stream, string directory)` — extracts the zip from the stream into the given directory:

  ```csharp
  ZipFileHelper.ExtractZipToDirectory(
      FileSystem.File.OpenRead("test.zip"), directory);
  ```

For a higher-level API, also see the [`Testably.Abstractions.Compression`](../companion-libraries#testablyabstractionscompression) companion package.
