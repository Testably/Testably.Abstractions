# Companion libraries

`Testably.Abstractions` ships two optional companion packages that extend `IFileSystem` with additional capabilities.

## Testably.Abstractions.Compression

Wraps the static methods on [`ZipFile`](https://learn.microsoft.com/en-us/dotnet/api/system.io.compression.zipfile) as an extension on `IFileSystem`.

```powershell
dotnet add package Testably.Abstractions.Compression
```

```csharp
IFileSystem fileSystem; // injected

fileSystem.ZipFile()
    .CreateFromDirectory("your-directory", "your-file.zip");

fileSystem.ZipFile()
    .ExtractToDirectory("your-file.zip", "your-destination");
```

[NuGet](https://www.nuget.org/packages/Testably.Abstractions.Compression) · See also: [Zip files example](./examples/zip-file)

## Testably.Abstractions.AccessControl

Implements the methods from [`FileSystemAclExtensions`](https://learn.microsoft.com/en-us/dotnet/api/system.io.filesystemaclextensions) on the `IFileSystem` interface.

```powershell
dotnet add package Testably.Abstractions.AccessControl
```

```csharp
IFileSystem fileSystem; // injected

DirectorySecurity accessControl = fileSystem.Directory.GetAccessControl("your-directory");
fileSystem.Directory.SetAccessControl("another-directory", accessControl);
```

[NuGet](https://www.nuget.org/packages/Testably.Abstractions.AccessControl) · See also: [ACL example](./examples/access-control-lists)
